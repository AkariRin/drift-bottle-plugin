# 漂流瓶

🍾 扔下漂流瓶，捡起属于你的惊喜。在群里和其他群友进行匿名互动交流。

## 功能特性

- 🎯 **扔漂流瓶**: 写下你想说的话，让它漂向远方
- 🎁 **捡漂流瓶**: 随机捡起一个漂流瓶，看看里面有什么
- 🔒 **匿名互动**: 保护用户隐私，营造轻松交流氛围
- ⏱️ **冷却机制**: 防止刷屏，保证体验

## 触发命令

### 扔漂流瓶
- `扔漂流瓶 [内容]` - 扔出一个带有内容的漂流瓶
- `扔瓶子 [内容]` - 同上

示例：
```
扔漂流瓶 今天天气真好！
扔瓶子 有人一起玩游戏吗？
```

### 捡漂流瓶
- `捡漂流瓶` - 随机捡起一个漂流瓶
- `捡瓶子` - 同上

## 配置说明

插件配置文件位于 `config.toml`：

```toml
[plugin]
name = "drift-bottle-plugin"  # 插件名称
enabled = true                 # 是否启用插件
config_version = "1.0.0"       # 配置版本

[settings]
throw_cooldown = 60            # 扔瓶子冷却时间（秒）
pick_cooldown = 30             # 捡瓶子冷却时间（秒）
```
